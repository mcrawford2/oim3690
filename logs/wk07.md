# What I learned
- Forms (sign up, log in, contact, search)
    - action - where to send data
    - method - GET (in URL) or POST (in body)
    - POST sends a lot more than just form inputs to the server.
- input types
    - text, email, password, number, date, color, range
- labels improve: screen reader support, click target, mobile usability
- meta name="viewport" content="width=device-width, initial-scale=1.0": without this line, everything looks tiny on mobile
- media quiries help with responsiveness
- devtools device mode: cntrl + shift + M
