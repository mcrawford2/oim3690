# Review
- ul: unordered (plain html), bullets
- li: list item, numbered
- img has no end tag (<img.../>)
- alt: makes website images accessible to be read

# What I learned
- line break: <br> breaker, creates line break in text and moves to next line
- <hr> horizontal rule, can be used to create a horizontal line
- <strong> makes text bold
- <em> emphasizes text
- <u> underlines text
- <i> italicizes text
- <div> division, includes a lot of information in one box and is used to divide pages into multiple boxes
- <script> refers to javascript
- <span> inline styling of text, selecting a part of a paragraph to make it look different

Semantic vs Non-semantic
- Semantic (preferred)
    <header>
        <nav>...</nav>
    <header>
- Non-semantic (ai generated)
    <div class = "header">
        <div class = "nav>...</div>
    </div>

Structure Rec
- header <header>

- main content <main>, followed by <article>,<section>,<div>
- side bar <aside>
